# home-venetian-hub

本仓库用于归档和发布多个独立 HTML 页面。所有页面均为静态内容，不依赖后端服务，可直接通过浏览器打开或部署至任意静态托管平台。

## 项目简介

`home-venetian-hub` 是一个集中存放各类 HTML 页面的仓库。每个页面独立存在，互不依赖。仓库本身不指向或服务于任何特定网站、域名或商业用途，仅作为页面存档与分享的容器。

## 目录说明

```
.
├── index.html          # 仓库入口页面（可选）
├── pages/              # 归档页面存放目录
│   ├── example-1.html  # 示例页面 1
│   ├── example-2.html  # 示例页面 2
│   └── ...
├── assets/             # 公共资源（如样式、脚本、图片）
│   ├── css/
│   ├── js/
│   └── images/
└── README.md           # 本文件
```

- `pages/`：所有独立 HTML 页面存放于此，文件名应反映页面内容主题。
- `assets/`：页面共享的静态资源，建议按类型分目录存放。
- `index.html`：可选的仓库首页，用于提供导航或说明。

## 页面归档说明

每个页面均为自包含的 HTML 文件，内部可引用 `assets/` 下的资源。归档页面应满足以下基本要求：

- 页面结构完整（含 `<!DOCTYPE html>`、`<html>`、`<head>`、`<body>`）。
- 资源引用使用相对路径（相对于仓库根目录）。
- 页面标题（`<title>`）应简要概括内容。
- 不包含外部追踪、广告或动态加载的外部脚本。

如需新增页面，请直接在 `pages/` 下创建 `.html` 文件，并在必要时更新 `index.html` 的导航链接。
